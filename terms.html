Política de Uso & Direitos

Última atualização: 11 de Junho de 2026

1. Sobre o Projeto

O projeto "Gift To Love" foi desenvolvido como uma demonstração interativa de amor, utilizando tecnologias modernas de front-end e bibliotecas de animação avançadas para entregar uma experiência imersiva e responsiva.

2. Direitos Autorais

Todos os direitos autorais sobre a estrutura narrativa, identidade visual personalizada e combinações de códigos (incluindo lógicas matemáticas para o motor de animações GSAP) apresentadas neste projeto são de propriedade exclusiva do autor (Cristofer). Todo e qualquer uso ou cópia de seções proprietárias requer autorização prévia.

3. Uso de Bibliotecas de Terceiros

Este site faz uso de ferramentas de código aberto e bibliotecas comerciais com licenciamento aplicável:

  • GSAP (GreenSock Animation Platform): Motor de animação central e ScrollTrigger, responsável pelas físicas de arraste e de rolagem. Todos os direitos da engine pertencem à GreenSock.
  • Fancybox: Biblioteca premium de visualização de mídia. Licenciada sob os termos da Fancyapps.
  • Lenis: Biblioteca para suavização universal de scroll, desenvolvida pelo Studio Freight (Código aberto).

4. Tratamento de Dados Pessoais

Este site é uma plataforma estática de visualização não comercial. Nenhuma informação pessoal do visitante é ativamente coletada, rastreada para fins de publicidade ou comercializada com terceiros.
